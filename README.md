# Crenexi Now
[now.crenexi.com](https://now.crenexi.com)

## NPM

#### Start

- `npm install`
- `npm start`

#### Lint

- `npm run lint`
- `npm run lint:fix`

#### Build

- `npm run build`
- `npm run build:dev`
- `npm run build:prod`

### Other

- `npm run stats` (create and analyze bundle stats)
- `npm run cleanup` (removes `dist`)
